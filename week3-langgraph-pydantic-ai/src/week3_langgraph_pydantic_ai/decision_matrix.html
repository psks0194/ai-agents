IntellAIgent · Week 3

Agent framework decision matrix

After building the same task in three implementations
If your workflow is
A single agent producing typed output, possibly with tools
Reach for
Pydantic AI
Function-as-agent mental model. Type hints become schemas. Loop is hidden. ~40% less code than LangGraph for this shape.
If your workflow is
Multi-step with branches, cycles, persistence, or human-in-the-loop
Reach for
LangGraph
Graph-as-workflow mental model. interrupt() and checkpointers are first-class. The verbosity is the price you pay for time travel and resumable state.
If your workflow is
You need to understand the protocol underneath the abstractions
Reach for
Raw SDK
~80 lines for a tool-using agent. Forces you to confront the loop. You'll never see frameworks the same way after writing the loop by hand.
The frameworks aren't competing. They're solving different shapes of problem.
